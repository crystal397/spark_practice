## 1. brew를 이용한 Java 설치

```python
$ brew update
$ brew tap adoptopenjdk/openjdk # adoptopenjdk/openjdk 레포지토리 추가
$ brew search jdk # 다운로드 가능한 jdk 버전 확인
$ brew install --cask adoptopenjdk11 # java 11 다운로드
$ java --version # 자바 버전 확인
```
